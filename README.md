# Xadrez Simples com HTML, CSS e JavaScript

Este projeto é um jogo de xadrez simples desenvolvido para o trabalho N2 de Oficina de Programação.

O objetivo principal do projeto é praticar a lógica em JavaScript, usando funções, eventos de clique, manipulação do HTML e controle dos dados do tabuleiro.

## Tecnologias usadas

* HTML
* CSS
* JavaScript

## Como funciona

O jogo possui um tabuleiro de xadrez 8x8, onde as peças são representadas por imagens.
O jogador clica em uma peça e depois clica na casa para onde deseja mover.

O sistema verifica se o movimento é válido de acordo com a peça escolhida.

## Funcionalidades

* Tabuleiro criado com JavaScript
* Peças usando imagens
* Seleção de peças pelo clique
* Movimentação básica das peças
* Troca de vez entre brancas e pretas
* Reinício da partida
* Vitória ao capturar o rei adversário

## Observação

Este não é um xadrez completo profissional.
O jogo não possui regras avançadas como xeque, xeque-mate, roque ou en passant.

A ideia foi fazer uma versão mais simples para praticar a lógica de programação com JavaScript.

## Como executar

1. Baixe ou clone este repositório.
2. Abra o arquivo `index.html` no navegador.
3. Clique nas peças para jogar.
